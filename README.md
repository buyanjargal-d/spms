# SPMS - Student Pickup Management System
# Сурагчдыг асран хамгаалагчид хүлээлгэн өгөх систем

Бакалаврын судалгааны ажил: DevOps-с DevSecOps руу шилжих ухагдахууны өөрчлөлт

## Зорилго
Сурагчдыг аюулгүй хүлээлгэн өгөх процессыг дижиталчлах, DevSecOps зарчмуудыг хэрэгжүүлэх

## Технологийн стак
- **Backend:** Node.js + Express.js + TypeScript + PostgreSQL
- **Frontend:** React.js + Tailwind CSS
- **Mobile:** React Native
- **Cache:** Redis
- **DevOps:** Docker + Docker Compose + GitHub Actions
- **Security:** SAST (SonarQube) + DAST (OWASP ZAP) + SCA (Snyk) + Trivy
